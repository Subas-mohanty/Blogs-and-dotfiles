### Here is the difference between Intel and ARM(Apple's 'M' chips) you need to know.

###### Intel chips works on x86 architecture, mostly desktop computers except apple macbooks. While ARM chips works in smartphones and macs.
###### Intel uses CISC(complex instruction set computing), which means it can perform multiple operations in a single command which makes it more powerful, more power consuming and high thermals.

###### While on the other hand ARM based chips use RISC(Reduced Instruction Set Computing), it performs simple and fixed length instructions which executes faster making it power efficient and good thermals.

#### How ARM chips are power efficient ?
- ARM chips has GPU and sometimes RAM integreted in them, its not like intel chips where the GPU and RAM are separate. This makes ARM chips power efficient as the data transfer between CPU and GPU is faster and less power consuming.
- Think of it like a fridge, where you have all off your food in one place, you don't have to go to different places to get your food, this makes it power efficient.

Modern intel chips uses micro-op translation which converts CISC instructions to RISC instructions, which makes it power efficient. And also ARM chips are getting more powerful as we see in Apple's M series chips and Snapdragon X Elite chips.
